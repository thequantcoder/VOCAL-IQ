# VocalIQ — User Guide

## Getting started
1. **Sign in** at your dashboard URL (or your reseller's white-labeled domain).
2. **Create an agent** — pick a template or start blank; set its persona, voice, and language(s).
3. **Design the flow** in the visual builder (say / listen / decision / tool / knowledge / transfer).
4. **Test** in the simulator, then **publish**.
5. **Deploy** on a phone number (Numbers), the web widget, or SIP.

## Running calls
- **Outbound / campaigns** — upload contacts, pick an agent, launch. DNC + consent + abuse controls
  are enforced automatically.
- **Inbound** — point a number's webhook at VocalIQ; the assigned agent answers.
- **Monitor** — live tiles on the dashboard; **Agent Desk** for human takeovers/transfers.

## Analyzing
- **Analytics** (volume, outcomes, sentiment, cost), **Latency** (p50/p95 vs SLO), **Search**
  (semantic transcript search), **QA scoring**.

## Billing
- **Wallet** (prepaid credits) or a subscription **Plan**; usage is metered per call (STT+LLM+TTS+telephony).
